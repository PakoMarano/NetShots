Aim: Design and develop a (1) multi-user application for smartphones that uses at least (2) sensors, (3) 2D graphics,(4) image processing, (5) cloud services and (6) concurrency.

Idea: NetShots (Social Padel Tracker)

NetShots is a mobile social application for the Padel community, designed to record match sessions, visualize performance statistics, and share enhanced photos with friends.

Requirement -> Feature in NetShots
(1) Multi-user Application -> Player Profiles & Following
(2) Sensors -> GPS Match Tracking
(3) 2D Graphics -> Statistical Visualization
(4) Image Processing -> Photo Filters & Enhancement
(5) Cloud Services -> Persistent Storage
(6) Concurrency -> Background Image Processing
